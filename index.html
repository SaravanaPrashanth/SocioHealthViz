Project Overview

Objective

Analyzing and predicting life expectancy using machine learning models, considering various socio-economic and health indicators.

Dataset

Comprehensive life expectancy data including socio-economic inequalities, mortality rate and deprivation decile across different regions(England and Wales) and periods.

Tools Used

R, tidyverse, caret, ggplot2, and various machine learning packages for model development and evaluation.

Data Science Pipeline

1. Data Processing

Initial data cleaning, handling missing values, and feature engineering performed in R/data-processing/nlt_processing.R and R/data-processing/sii_processing.R

2. Exploratory Analysis

Comprehensive EDA using visualization techniques implemented in R/visualizations/visualization_plots.R

3. Statistical Analysis

Statistical tests and correlation analysis conducted in R/analysis/avoidable_mortality_analysis.R and R/analysis/correlation_analysis.R

4. Model Development

Machine learning models built and evaluated in R/model/predictive_models.R
