Project Overview

Objective

Analyzing and predicting life expectancy using machine learning models, considering various socio-economic and health indicators.

Dataset

Comprehensive life expectancy data including health factors, economic conditions, and social indicators across different regions and time periods.

Tools Used

R, tidyverse, caret, ggplot2, and various machine learning packages for model development and evaluation.

Data Science Pipeline

1. Data Processing

Initial data cleaning, handling missing values, and feature engineering performed in `R/data-processing`

2. Exploratory Analysis

Comprehensive EDA using visualization techniques implemented in `R/visualizations`

3. Statistical Analysis

Statistical tests and correlation analysis conducted in `R/analysis`

4. Model Development

Machine learning models built and evaluated in `R/model`

Key Findings

Feature Importance

  • We identified key factors that affect life expectancy, such as health conditions and living standards.
  • Key factors include death rates and the level of poverty in different areas.
  • These factors help explain why life expectancy can vary greatly between different communities.

Predictions

  • We tested how accurately our model can predict life expectancy across regions of England and Wales.
  • The model that we used showed much better predictions than other simpler methods.
  • Our model was able to make reliable predictions about life expectancy trends based on past data.
